在linux下利用wine运行RTX

2011-10-23
技术

环境：fedora 15 / Wine

tips：在fedora上安装wine很简单。只需要打开终端，输入wine然后回车。系统就会问你是否安装wine软件包以便提供此命令。选择yes就好了。对于大多数的常用软件，在fedora下都可已用这种方法安装。也可以利用 yum install wine命令来安装（在root用户下）。

步骤：

  1. 打开终端，依次运行下列命令（某些命令可能需要root权限）：

  2. 去这里下载一个名为winetricks的脚本，这个脚本主要是帮助下载一些wine中常用的加载项。

  3. 运行如下命令： sh winetricks msxml3 gdiplus riched20 vcrun6 vcrun2005sp1 vb6run 。这个命令下载rtx需要的一些加载项。其中msxml3无法下载，请在网上搜索msxml3，放对cache（.cache/winetricks/mscml3/） 目录下。其他的类似。你可以根据脚本的提示进行操作。

  4. 去腾讯的官网下载RTX的安装包。

  5. 运行 wine rtxclient2010formal.exe 命令

  6. 从xp系统里拷贝一份宋体字体到 ~/.wine/drive_c/windows/Fonts/ 文件夹下。这个可以解决rtx无法显示中文问题

9.在应用程序列表中找到 Wine Configuration，运行之。选择“函数库”选项卡，新添加oleaut32.dll函数库，使其原装先于内建。这可以解决运行RTX后列表中的中文名字显示不全的问题。

ok ，现在你已经可以比较完美的运行RTX了。fedora的话在应用程序里就可以看到rtx的启动图标。

注意，登录rtx后，你会发现一直处于离开状态。这时请在设置中将自动更改为离开状态的复选框去掉。

在fedora下（事实上在大多数linux下都有这些问题），你还要注意一下问题。不要点击最大化按钮，一旦你点击了最大化按钮，你将无法恢复原状，除非关闭聊天窗口。不要试图查看历史消息。这会引起程序崩溃而退出。你可以点击聊天窗口的历史消息按钮来查看一小段时间内的聊天记录。还有，快捷键在全局下是不起作用的，只有在rtx的任意一个窗口获得焦点的时候，快捷键才起作用。
